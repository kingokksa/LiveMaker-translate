# LiveMaker-translate

# 要求

* 游戏
* 安装了pylivemaker插件的Translator++ （解包、提取文本）
* 任意Font Obfuscation 工具 （字体混淆与生成，防止字库缺失）
* Sakura或任意可破限翻译的本地或在线模型 （机翻模型）
* AI translator 等可以兼容各类引擎文本导出格式的翻译器（调用模型翻译）
* FontMod （Hook游戏字体，解决部分字体不匹配问题）
